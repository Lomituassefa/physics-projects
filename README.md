# Harmonic Oscillator Simulation

A Python simulation of a mass-spring system built as part of my physics + computational science learning journey.

## What it simulates

- **Act 1 — Pure Oscillation**: A mass on a spring with no friction. Oscillates forever. Solves the equation x'' + ω²x = 0
- **Act 2 — Damped Oscillation**: Three damping cases showing underdamped, critically damped, and overdamped motion
- **Act 3 — Resonance**: Driving the system at different frequencies. When driving frequency matches natural frequency — amplitude explodes

## Tools used
- Python 3
- NumPy
- Matplotlib

## Physics concepts covered
- Newton's second law
- Ordinary differential equations
- Energy conservation
- Resonance phenomena

## How to run
```bash
python3 oscillator.py
```
